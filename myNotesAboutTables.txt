Класс TableConfiguration

Задачи:
  1) задать table settings
  2) задать параметры пейджинации
  3) задать стили для body, header, footer

  - методы:
    columnSet(array $set) - задает поля выводимы в таблице и их порядок
        аргумент: массив - названия полей в sql таблице. Порядок вывода полей соответствует порядку в массиве

    для каждого поля задаем набор свойств:
        title - текст для вывода в хедере таблицы
        description - описание (всплывает при наведении на заголовок)
        width - ширина колонки. если без 'px', то значит задана в процентах
        sortable -
        filterable -
        id - если не задан, соответствует имени поля
